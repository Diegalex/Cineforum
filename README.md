# Práctica 2.5
Entrega de la práctica 2.5 de Desarrollo de Interfaces. 

Página para la compra de entradas de un cineforum en el fichero "cineforum.html", el cuál hace uso de archivos complementarios de estilo CSS y de script JS del mismo nombre además de imagenes para cada película bajo la carpeta "imagenes".

En la página seleccionas el tipo de entrada mediante un desplegable, seleccionas la cantidad de entradas y la sesión de la peli a la que quieres ir; con esos datos se rellena el formulario, donde el usuario únicamente introduce nombre, apellidos y su correo eléctronico. Se comprueba que los datos introducidos son válidos y en caso de estar correcto el formulario se genera un resumen de tu compra con el precio total final, y en caso de que algún dato sea erroneo se mostrará una alerta con cada campo erroneo.
